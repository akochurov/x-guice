Classes with @DB annotated methods should:
  • Have an EntityManager field
  • EntityManager field should not be static
Usage of EntityManager field allowed only in @DB annotated methods.
If @DB annotated method use EntityManager for persist, merge or remove operations, transaction support must be enabled, else using of transaction support is not required and could be the cause of performance problems.
Methods joinTransaction() and getTransaction should not be used into @DB annotated methods. Use @DB(transaction=REQUIRED) instead.
